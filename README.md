# Flask REST API Project

## Features
1. CRUD operations(CREATE, READ, UPDATE, DELETE)
2. Input Validation
3. MySQL database integration

## Tech Stack
1. Python
2. Flask
3. MySQL

## Endpoints
- GET /users
- POST /users
- PUT /users/<id>
- DELETE /users/<id>

## How to run
1. Install dependencies
2. Run app.py
3. Use Postman to test APIs
